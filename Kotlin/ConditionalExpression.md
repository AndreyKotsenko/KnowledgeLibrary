# Условные выражения
### if-else конструкция
В основном эта конструкция работает как и в Java.
Пример:
```
if (number > 10 $$ number < 15)
  println("Value between 10 and 15")
else if ( number == 10 || number == 15)
  println("Value: 10 or 15")
else 
  println("Something else")
```

Но так же есть одна прикольная конструкция в Kotlin. Мы можем передать значение из условного выражения в переменную:
```
var res = if(number > 30) 1 else 0
```

### Конструкция when
**When** это аналог конструкциз из Java, такой как **Switch**. 
Пример:
```
when(number) {
  null -> println("...")
  10 -> println("...")
  else -> println("...")
  }
```
То есть сначала идет перечисление возможных вариантов, а если не один не подошел то выполняется действие из блока **else**.

Так же переменной можно присвоить результат выполнения этого выражения:
```
var words = when(number) {
     null -> "Nothing"
     else -> number
     }
```
При этом тип данных переменной заранее не определен. В данном случае он может быть как **String** так и **Int**. Все зависит от результата выполнения даного выражения.
