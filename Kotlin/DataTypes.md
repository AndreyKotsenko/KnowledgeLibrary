# Data Types and Variables
В Kotlin все типы данных такие же как и в Java: Byte, Short, Int, Long, Double, Float, Char, String, Boolean.

Но обьявление переменных отличается. Их можно обьявлять двумя способами. 
 
 - с указанием типа данных
 - без указания типа данных

### С указанием типа данных
Конструкция: 
```
var name: Type = values
```
Пример:
```
var number: Int = 10
```

### Без указания типа данных

Пример:
```
var number = 10
```

В таком случае тип данных определяется автоматически.

Так как Kotlin работает по принципу статической инициализации переменной, то после определения типа данных для переменой, его уже нельзя будет сменить.

### Константы

Константы это переменные значение которых нельзя менять после инициализации.

Обьявляються они так же как и обычные переменные только вместо ключевого слова **var** мы используем **val**.

Пример:
```
val number: Int = 10
```

### NULL значение
В Kotlin нельзя просто так присвоить значение NULL. Это специально сделано что бы избежать частых ошибок Null Pointer Exceprion как в Java.

Если мы хотим использовать NULL значение для нашей переменной то нам нужно использовать знак **"?"**

Пример:
```
var word: String? = null
```

### Математические операции
Все математические операции такие же как и в Java: + - * / % += -= *= /= %= 

Так же есть инкрементация и декрементация:

variable++

variable--

++variable

--variable
