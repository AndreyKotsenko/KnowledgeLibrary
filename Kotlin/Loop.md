# Циклы
Основная функция циклов, такая же как и во всех языках, повторение определных действий описанных в блоке цикла, определенное количество раз.

### while
Цикл **While** работает пока заданное условие остается верным. Пример:
```
    while(items > 0){
        println(items)
        items--
    }
```

### do-while
Цикл **do-while** работает так же как и **while** только вне зависимости от условия, даже если оно сразу не соблюдается, то один раз блок с действиями все равно будет выполнен. Пример:
```
    do {
        println("Items: $items")
    } while(items == 5)
```


### for
Цикл **for** уже явно задает такие параметры как количество итераций, шаг итерации и т д. В Kotlin он утроен лучше чем в Java. Пример:
```
    for(i in 0..3) {
        println(i)
    }
```
В данном случае цикл будет проходить три итерации, так как мы задали **range** от 0 до 3.

Так же мы можем задать шаг итерации. Пример:
```
    for(i in 0..10 step 2) {
        println(i)
    }
```

Так же мы можем сделать реверсию. То есть наша переменная **i** будет начинаться с конечного значения нашего **range** и заканчиваться начинающим значением. Пример:
```
    for(i in 10 downTo 0 step 2) {
        println(i)
    }
```

Примечательно то что так же в таких цыклах можно переберать алфавит. Пример:
```
    for(el in 'a'..'d') {
        println(el)
    }
```

Для алфавита так же можно менять шаг и делать реверсию. Примеры:
```
    //loop with alphabet and step
    for(el in 'a'..'y' step 2) {
        println(el)
    }

    //reverse loop with alphabet and step
    println("Down alphabet")
    for(el in 'z' downTo 'a' step 5) {
        println(el)
    }
```


### if
Примечательно то что мы можем использовать **range**  для условного оператора **if**. В таком случае если значение попадает в последовательность то действие выполняется иначе нет. Примеры на проверку вхождения в последовательность и наоборот:
```
    val x = 20

    //range in IF
    if(x in 5..30) {
        println("Variable: $x")
    }

    if(x !in 5..30) {
        println("Variable: $x")
    }
```


Так же есть еще циклы по типу **for-each** и другие. С ними можно ознакомится в блоке **массивы**.

### Операторы continue и break

Иногда при использовании цикла возникает необходимость при некоторых условиях не дожидаться выполнения всех инструкций в цикле, перейти к новой итерации. Для этого можно использовать оператор continue:
```
for(n in 1..8){
    if(n == 5) continue;
    println(n * n)
}
```
В данном случае когда n будет равно 5, сработает оператор continue. И последующая инструкция, которая выводит на консоль квадрат числа, не будет выполняться. Цикл перейдет к обработке следующего элемента в массиве

Бывает, что при некоторых условиях нам вовсе надо выйти из цикла, прекратить его выполнение. В этом случае применяется оператор break:
```
for(n in 1..5){
    if(n == 5) break;
    println(n * n)
}
```
В данном случае когда n окажется равен 5, то с помощью оператора break будет выполнен выход из цикла. Цикл полностью завершится.
