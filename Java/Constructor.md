# Конструктор

Конструктор (Constructor) в Java - это специальный метод, который вызывается при создании экземпляра (объекта) класса. Он используется для инициализации свойств объекта и подготовки его к использованию.

Конструктор имеет имя, совпадающее с именем класса, и может принимать параметры. Он вызывается автоматически при создании нового объекта с помощью оператора "new" и может выполнять любой код, необходимый для инициализации объекта.

Если в классе не определен конструктор, то Java создает для него конструктор по умолчанию без параметров, который не выполняет никаких действий, кроме создания объекта.

Если в классе определен хотя бы один конструктор, то конструктор по умолчанию не создается автоматически.

Конструкторы могут быть перегружены, то есть класс может иметь несколько конструкторов с разными параметрами. В этом случае при создании объекта необходимо указать нужный конструктор в зависимости от требуемых параметров.

Использование конструкторов позволяет обеспечить правильную инициализацию объектов, что является важным аспектом объектно-ориентированного программирования.

Если наш класс унаследован от родительского то мы должны вызвать родительский конструктор в дочернем используя **super()**.

Пример:
```
package com.company.akotsenko;

public class ConstructorExample extends Parent{
    int i;

    ConstructorExample(){
        super("Max");
        System.out.println("I am created");
    }

    ConstructorExample(String hello, int i) {
        super("Andrey");
        this.i = i;
        System.out.println(hello);
    }

    ConstructorExample(String name) {
        super(name); // called constructor from Parent/ it is required
    }
}


class Parent {
    Parent(String name) {
        System.out.println("Parent " + name);
    }
}
```

Здесь мы создали 2 класса. В классе **ConstructorExample** мы создали 3 конструктора которые в свою очередь вызывают родительский конструктор через **super()**.

Так же если вы хотите вызвать из конструктора другой конструктор в том же классе, то можно использовать **this()**.

Main функция:
```
public class Main {

    public static void main(String[] args) {
        ConstructorExample test = new ConstructorExample();

        ConstructorExample test2 = new ConstructorExample("Hi", 5);

        ConstructorExample test3 = new ConstructorExample("Andrey");

    }
}
```
