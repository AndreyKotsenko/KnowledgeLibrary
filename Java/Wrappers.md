# Обертки

В Java обертки (wrapper) — это классы, которые инкапсулируют примитивные типы данных (например, int, double, boolean) в объекты. Каждый примитивный тип имеет соответствующий ему класс-обертку:

Boolean для boolean
Character для char
Byte для byte
Short для short
Integer для int
Long для long
Float для float
Double для double
Классы-обертки предоставляют различные методы для работы с примитивными типами данных в объектной форме. Например, Integer класс содержит методы для конвертации строк в целые числа (parseInt) и для конвертации целых чисел в строки (toString).

Обертки также используются при работе с коллекциями объектов, так как в Java коллекции могут хранить только объекты, а не примитивные типы. При необходимости использовать примитивные типы в коллекциях, необходимо использовать соответствующие классы-обертки.

Так же важным фактором является что обертки, так же как и другие ссылочные обьекты нужно сравнивать через equals(), а не через "==", так как первое сравнивает именно значение, а второе по памяти.
