# Data types

В Java есть 8 примитивных типов данных: byte, short, int, long, float, double, boolean, char. Остлаьные переменные будут считаться ссылкочными.

Типы данных нужны что бы мы могли определить что будет хранить переменная (символ, число целое, вещественое число) и сколько байт будет для этого использовано.

Пример примитивных типов данных:
```
    byte b = 0; //1 byte(256) -128 ... +128
    short s = 1234; // 2 byte -32k .... 32k
    int i = 23232352; // 4 byte
    long l = 3422323; // 8 byte
    float f = 12.324f; // 4 byte
    double d = 1234.3252352; // 8 byte
    boolean bool = false;
    char c = 'c';
```

# Packages

В Java package (пакеты) используются для организации классов в логически связанные группы. Это позволяет более удобно управлять классами, упрощает их переиспользование и обеспечивает более чистую структуру программы.

Package может содержать в себе один или более классов, интерфейсов, подпакетов, а также файлы свойств и другие ресурсы, которые используются в приложении.

Когда вы создаете классы внутри package, вы можете использовать ключевое слово package для указания названия package в начале файла с кодом класса. Это позволяет другим классам в вашей программе использовать классы из этого package, используя полное имя класса, которое включает имя package.

Например, если у вас есть класс MyClass, который находится в package com.example, то другой класс может использовать этот класс, используя полное имя класса com.example.MyClass.

Таким образом, использование package в Java упрощает структурирование кода, обеспечивает лучшую организацию, уменьшает конфликты имен и облегчает повторное использование кода.

Для того что бы использовать какой-то функционал из другого пакета то нужно использовать ключевое слово **import**.

Пример:
```
import com.example.*
```

В данном примере мы имортируем весь функционал из пакета **com.example**
