# Наследование

Наследование в Java - это механизм, который позволяет классам наследовать свойства и методы других классов. Класс, который наследует свойства и методы, называется подклассом, а класс, свойства и методы которого наследуются, называется суперклассом или родительским классом.

Для создания подкласса в Java используется ключевое слово extends, после которого указывается имя суперкласса. Подкласс наследует все поля и методы суперкласса, за исключением приватных методов, которые не доступны из подкласса. Подкласс может также добавлять свои собственные поля и методы, а также переопределять методы суперкласса, если это необходимо.

Пример:
```
public class Car {

    int speed = 100;

    int getSpeed(){
        return speed;
    }

    private String getSubject(){
        return "It is car";
    }
}
```

В этом примере мы создали класс Car с полем скорость и методами для получения скорости и просто тестовым приматным методом который возвращает строку.

```
public class Toyota extends Car{

    int age = 10;

    int getAge(){
        return age;
    }
}

```

После чего мы создали класс который наследуется от класса машина, и при этом определяет свои поля и методы.

```
    public static void main(String[] args) {
        Toyota a = new Toyota();

        Car b = new Toyota();

        System.out.println(a.getAge());
        System.out.println(a.getSpeed());
        //System.out.println(a.getSubject()); error Because we can't access private fields and methods even with inheritance

        System.out.println(b.getSpeed());
        //System.out.println(b.getAge()); error 'cause class Car don't have method getSpeed
    }
}

```

Для примера создаем 2 обьекта. Первый это обьекта нашего класса Toyota. Как результат он имеет доступ к своим полям и методам, а так же к полям и методам родительского класса, помимо приватных.

Интересная особенность в том что мы можем создать обьект родительского класса но поместить туда дочерний класс. В таком случае мы будем иметь доступ только к методам и полям родительского класса, помимо приватых.
