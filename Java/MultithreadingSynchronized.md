# Синхронизация

Ключевое слово synchronized в Java используется для обеспечения синхронизации доступа к общим ресурсам, таким как переменные, методы или блоки кода, между несколькими потоками выполнения.

Когда метод или блок кода помечается ключевым словом synchronized, то только один поток может выполнять данный метод или блок кода в любой момент времени. Это гарантирует, что другие потоки не смогут войти в этот метод или блок кода и изменять общие ресурсы одновременно с другим потоком, что может привести к ошибкам и непредсказуемому поведению программы.

В Java ключевое слово synchronized может быть применено к методам, статическим методам, конструкторам и блокам кода. Кроме того, существует механизм блокировки монитора, который позволяет синхронизировать доступ к общим ресурсам между потоками.

Пример использования synchronized для синхронизации доступа к методу:
```
public synchronized void doSomething() {
    // Код метода
}
```

В данном примере метод doSomething будет синхронизирован, что означает, что только один поток может выполнить данный метод в любой момент времени. Если другой поток попытается вызвать этот метод в то время, когда первый поток еще не завершил выполнение, он будет заблокирован до тех пор, пока первый поток не освободит монитор.
